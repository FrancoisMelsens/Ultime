Ce site web permet de localiser ou des rechercher des stations de la RATP.
Il a été conçu pour s'adapter au différents formats de la fenêtre de l'utilisateur ce qui le rend utile pour les appareils mobiles.

Fonctions disponibles :

-----Recherche d'une station par nom----------
L'utilisateur peut rechercher une station de la RATP en saisissant son nom 

-----Recherche des stations proches----------
Grâce à la géolocalisation, l'utilisateur peut savoir où sont les stations les plus proches de son emplacement actuel.
Il peut aussi augmenter la distance de recherche, les distances que nous proposons sont 50, 100, 200, 500 et 1000 mètres.

Récupération des données :

Les données exploitées par l'application sont récupérées directement sur le site OpenDataSoft.

Les technologies employées sont :

-Angular
-Bootstrap
-Google Maps API
-Yeoman
